ТЕГИ:

1) html - Основной тег html

2) head - Шапка сайта, служащая для описания страницы сайта. Находящееся в этом теге не видно конечному пользователю, за исключением названия страницы, если вы его указываете

3) title - Тег, указывающий название страницы (пишется внутри тега head)

3) title - Тег, указывающий название страницы (пишется внутри тега head)

4) meta - Тег, позволяющий устанавливать различные мета-данные. Тег meta - одинарный тег (пишется внутри тега head)

5) body - Тег, содержимое которого доступно для обозрения конечным пользователям

6) h1 - Тег для заголовков. В зависимости от числа в конце, определяется размер текста в заголовке

7) p - Тег для создания текста-обзаца

8) b - Тег для жирного почеркивания текста

9) strong - Тег для жирного почеркивания текста

10) em - Тег для выделения текста курсивом

11) i - Тег для выделения текста курсивом

12) strike - Тег для выделения текста перечеркиванием

13) sub - Тег для уменьшения текста и его помещения ниже основного ряда (H2O)

14) sup - Тег для уменьшения текста и его помещения выше основного ряда (а2)

15) br - Тег переноса строки. Не оставляет отступы между строк. Является одинарным

16) hr - Тег, создающий горизонтальную линию. Является одинарным

17) abbr - Тег для обозначения аббревиатур (можно добавить атрибут title для добавления обозначения к аббревиатуре при наведении курсора мыши)

18) address - Тег для обозначения адреса (текст выделяется курсивом)

19) big - Тег для увеличения текста

20) small - Тег для уменьшения текста

21) pre - Тег для отображения текста с сохранением всех отступов (моноширный шрифт)

22) code - Тег для обозначения кода (моноширный шрифт)

23) ol - Тег для создания пронумерованного списка. Элементы списка указываются в тегах li

24) ul - Тег для создания непронумерованного списка. элементы списка указываются в тегах li

25) a - Тег для создания ссылок (должен быть обязательный атрибут href)

26) img - Тег для работы с изображениями. Работает только с атрибутами. Является одинармным.

27) table - Тег для создания таблиц. Может состоять из тегов thead, tbody, tfoot. Тег tr служит для создания строк в таблице и может использоваться как в теге table, так и в его внутренних тегах thead, tbody и tfoot (особой разницы нет, но правильнее использовать разделение создания таблицы на 3 части). Тег th создаёт в таблице столбец и используется внутри тега thead, и текст в столбце почеркивается жирным. Тег td создаёт в таблице столбец и используется внутри тега tbody и tfoot и не почеркивается жирным

28) link - Тег, позволяющий подключать css файлы, работает только с атрибутами, является одинармным, записывается только в теге head

29) script - Тег, позволяющий подключать java script файлы, работает только с атрибутами, является одинармным, рекомендуется ставить перед окончанием тега body

30) div - Тег, создающий блок, в котором можно будет подключать различные стили

31) span - Тег, выделяющий определенную часть кода, к которой можно будет обраться через язык css через этот тег

32) form - Тег для собрания в себя других тегов, предназначенных для работы с формочками:

  • label - Тег, создающий подпись для какого-либо поля
  • input - Тег, используемый для создания интерактивных элементов управления в веб-формах для получения данных от пользователя. Является одинармным тегом и имеет обязательный атрибут type

АТРИБУТЫ:

Тег meta:

1) charset - Атрибут, устанавливающий кодировку

Тег table:

1) border - Атрибут, устанавливающий границы в таблице. Значением для атрибута является число (количество пикселей в перегородке)

2) width - Атрибут, устанавливающий ширину таблицы в пикселях

3) align - Атрибут, устанавливающий текст в таблице (справа, слева, по центру)

Тег input:

1) type - Атрибут, устанавливающий тип поля

2) placeholder - Атрибут, устанавливающий подсказку в водимое поле

3) value - Атрибут, изначальный устанавливающий текст в поле

4) maxlength - Атрибут, устанавливающий ограничение на количество вводимых символов в поле

5) name - Атрибут, устанавливающий функциональное название для поля

6) id - Уникальный идентификатор для какого-либо объекта

Тег label:

1) for - Атрибут, указывающий с каким инпут объектом работает подпись (label)

СТИЛИ CSS:

1) color - Выбор цвета

2) text-align - Размещение текста

3) font-size - Размер текста

4) font-style -

5) font-family - Выбор шрифта

6) text-transform - Выбор капса

7) word-spacing - Расстояние между словами

8) letter-spacing - Расстояние между буквами

9) line-height - Расстояние между строками

10) text-derorating - Линия текста (под ним, над ним и тд.)

11) cursor - Состояние курсора при наведении

12) float - Позиционирование блоков. Способность поставить блоки на 1 линии

13) padding - Установка внутренних отступов блока

14) margin - Установка внешних отступов блока

15) border - Установка обводки

15) background - Установка цвета заднего фона

15)width - Установка ширины

15) height - Установка высоты

15) transition-delay - Стиль, задерживающий начало перехода. Секунды или милисекунды.

15) transition-property - Стиль, в который записывается объект, которому нужен плавный переход

15) transition-duration - Стиль, в который записывается в секундах или милисекундах скорость перехода

15) transition - Сокращение использования предыдущих стилей, например: transition: background 0.3s ease 1s

15) transform - Стиль, перемещающий объекты в пространстве, функции тега:

  • translate - Перемещение объекта по осям(первое значение для X, второе для Y)

  • perspective - Определяет расcтояние между плоскостью z = 0 и пользователем для того чтобы придать 3D-позиционируемому элементу эффект переспективы

  • translate3d - Объеденение функций translateX, translateY, translateZ в одну функцию(translate3d(x, y, z))

  • scale - Увеличение объекта

  • scale3d - Увеличение объекта

  • skew - Наклон элемента

  • rotate - Поворот объекта

  • rotate - Поворот объекта
